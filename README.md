# Тренажёр обратной связи

Обучающий инструмент для отработки навыка корректирующей обратной связи по методике:
позитив → ошибка/последствия → рекомендация.

## Функции
- 28 реальных кейсов
- 3 уровня сложности
- Автоматическая проверка по 7 критериям

## Запуск
Просто откройте index.html в браузере
