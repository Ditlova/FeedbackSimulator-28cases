# HR-game
Игра для HR о компетенциях.Это обучающая игра "HR Мастер" — тренажёр по компетенциям для HR-специалистов с 5 уровнями, системой звёзд, авторизацией и таблицей лидеров. Код написан для Firebase (Google Canvas) использована Gemini Авторизация (локальная, localStorage): Экран входа с полями ФИО и пароль Прогресс привязан к пользователю 5 уровней обучения: Hard и Soft Skills — drag-and-drop классификация навыков Классификация компетенций — найти лишнюю компетенцию в группе Компетенции и Функционал — выбрать критически важные компетенции для задачи Поведенческие индикаторы — определить компетенцию по описанию поведения Gрофиль должности — составить профиль из правильных компетенций Игровые механики: Система звёзд (3/2/1 в зависимости от попытки) Подсказки при ошибках с перемешиванием заданий Блокировка следующего уровня до прохождения текущего Подробный разбор после успешного прохождения Таблица лидеров (все зарегистрированные пользователи из localStorage)
